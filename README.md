## Who is this for?

This is for the nine of us who do notice how often a ⇧← comes out wrong, as if it were a plain ←, or something else

An example

    macOS Bash

    $ : # Press ⇧ ← to see ;2D and press ⇧ → to see ;2C

    Linux Bash, or macOS Zsh

    $ : # Press ⇧ ← to see D and press ⇧ → to see C
    $

I want consistent output for consistent input. And we know how to deliver
